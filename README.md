Требования к ОС для запуска программы:

Дистрибутив: Ubuntu 24.04.3 LTS.
Компилятор: g++ (Ubuntu) 13.3.0.
Требуется поддержка сокетов POSIX (код использует <sys/socket.h>, <arpa/inet.h> и т.п.), поэтому под Windows нативно код не соберётся без среды типа WSL/Cygwin/MSYS2


Инструкция по сборке:
Для сборки необходимо скомпилировать файлы: main.cpp, magma.cpp, file_utils.cpp, cbc.cpp командой:

g++ main.cpp magma.cpp cbc.cpp file_utils.cpp kdf.cpp -o magma.exe
Далее запустить magma.exe введя в консоль 
./magma.exe <режим работы> 


Режим работы:
Клиент-серверный режим
Программа может работать как сервер, который отправляет подготовленное зашифрованное сообщение, или как клиент, который принимает и расшифровывает его.
Доступны следующие команды:
1.	./magma --server
Запускает сервер, который читает сообщение из файла message.txt, шифрует его и ждёт подключения клиента.
Для работы требуется файл keys.txt в текущей директории.
2.	./magma --client <ip>
Запускает клиент, который устанавливает соединение с сервером, принимает от него зашифрованный пакет и выполняет расшифровку.
Клиент также использует файл keys.txt, который должен совпадать с серверным.
